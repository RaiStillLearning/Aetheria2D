# 🎮 2D Platformer Jump Game

Game platformer 2D sederhana dengan mekanik utama **lompat (jump)**. Project ini cocok untuk belajar dasar pembuatan game seperti physics, collision, dan kontrol karakter.

---

## 📌 Fitur

- 🏃 Gerakan karakter (kiri & kanan)
- ⬆️ Mekanik lompat (jump)
- 🧱 Collision dengan platform
- 🎯 Sistem gravity sederhana
- 🔁 Restart game

---

## 🛠️ Teknologi

- Engine: Unity 6000.3.2f1
- Bahasa:
  - C# (Unity)
